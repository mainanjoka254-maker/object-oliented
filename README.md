# Ian Maina - Developer Portfolio

A modern, responsive portfolio website built with HTML, CSS, and Object-Oriented JavaScript.

## Features

- **Fully Responsive** - Works seamlessly on all devices (desktop, tablet, mobile)
- **Object-Oriented JavaScript** - Clean class-based architecture with ES6+ features
- **Smooth Animations** - Scroll-triggered animations and skill bar progress
- **Contact Section** - Direct line and WhatsApp contact at **+254 140 844 495**
- **Dynamic Project Showcase** - Projects rendered programmatically via JavaScript
- **Mobile Navigation** - Hamburger menu with smooth transitions
- **Scroll Spy** - Active navigation link updates on scroll
- **Back to Top** - Floating button for quick navigation

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, CSS Variables, Animations)
- JavaScript (ES6+ Classes, Modules, Intersection Observer)

## JavaScript Classes

| Class | Responsibility |
|-------|---------------|
| `Navigation` | Navbar scroll effects, hamburger menu, scroll spy |
| `ProjectRenderer` | Renders project cards from data dynamically |
| `ScrollAnimator` | Fade-in animations on scroll via IntersectionObserver |
| `SkillBarAnimator` | Animated skill progress bars on viewport entry |
| `ContactForm` | Form validation and submission handling |
| `BackToTop` | Floating back-to-top button |
| `DynamicFooter` | Auto-updating copyright year |
| `PortfolioApp` | Main bootstrapper that initializes all modules |
